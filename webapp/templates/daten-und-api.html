Daten & API

Eine Informationsplattform ist immer nur so gut wie die ihr zugrunde liegenden Daten. Daher gleichen wir stündlich alle Daten mit den Datenquellen ab. Bei Free-Systemen streben wir noch eine schnellere Aktualisierung an, da wir aber die Server der Anbieter nicht überlasten wollen, wurde dies zunächst zurückgestellt.

Datenquellen

Die Daten auf dieser Website werden wenn möglich via offener API oder als offener Datensatz importiert. Wenn dies nicht möglich ist, werden die APIs der Buchungssysteme verwendet. Liegt selbst das nicht vor, wird auf einen Scraper zurückgegriffen. Folgende Datenquellen werden zur Zeit verwendet:

Außerdem werden für die Regions-Suche die Gemeinden aus dem Gemeindeverzeichnis-Informationssystem GV-ISys des Bundesamtes für Statistik verwendet.

Schnittstelle (API)

Über die JSON-Schnittstelle können die auf der Seite präsentierten Daten abgegriffen werden. Aber Achtung: die Schnittstelle ist noch nicht stabil, da das Datenmodell noch nicht final ist. Die Stationen können abgerufen werden unter:

/api/providers

/api/stations

Die ausgelieferten JSON-Objekte ist weitestgehend selbsterklärend. Unter next erhält man die nächsten 100 Objekte. Der Wert null bedeutet, dass (noch) keine Daten vorliegen.

Fehler melden

Das Projekt wurde gerade erst gestartet, so dass noch einige Daten fehlen und einige Fehler zu finden sein dürften. Um Fehler zu melden oder Erweiterungen vorzuschlagen, nutzen Sie bitte den Bugtracker auf Github.
